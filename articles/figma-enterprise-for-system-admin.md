# 1. はじめに
Figmaはブラウザベースのコラボレーション・インターフェイス・デザイン・ツールです。
今やトップのシェア率を誇り、UIデザイナーにとって必要不可欠のツールになりつつあります。

Figmaは非常に強力なツールですが、システム管理者からみると独自の設計方針に頭を悩ませることが多いことも事実です。
この記事ではFigmaエンタープライズプランをシステム管理者向けに解説します。
